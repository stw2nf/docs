### Size and Weight

| Item                                  | Value                                    |
| ------------------------------------- | ---------------------------------------- |
| Dimensions (L x W x H)                 | 63" x 35.5" x 18"<br />(5.25 x 2.95 x 1.5 ft) |
| Base Weight<br />(including 25 AH Battery) | 700 lbs                                  |
| Track Width                           | 7"                                       |


### Speed and Performance

| Item        | Value     |
| ----------- | --------- |
| Max Payload | 2,200 lbs |
| Max Incline | 50 deg    |
| Max Speed   | 4 mph     |

### Battery and Power

| Item              | Value         |
| ----------------- | ------------- |
| Battery Chemistry | LiFePO4       |
| Fuel Tank         | 1 gal (gas)   |
| Charge Time       | 1.5 hrs       |
| Power             | 5,000 W (48V) |

| Item       | Value                                                                         |
| ---------- | ----------------------------------------------------------------------------- |
| Control    | MavCommander Device and XBox Style RC Controller                              |
| Base Range | 600ft (Infinitely Extensible Mesh Network)                                    |
| Accuracy   | Centimeter Level RTK GPS Accuracy                                             |
| Sensing    | Forward Facing Depth Sensing Camera, 2 x 3D Lidar for 360deg Obstacle Sensing |
