| Number | Name                         | Description                                 |
| ------ | ---------------------------- | ------------------------------------------- |
| 1      | Activate / Deactivate Button | Push button to activate and deactivate PAL  |
| 2      | Exit Manual Control Button   | Exit manual control and return to homepage  |
| 3      | Throttle Slider              | Control forward and reverse throttle of PAL |
| 4      | Steering Slider              | Control right and left steering of PAL      |
