### Saving Key Locations

Saving key locations allows for locations to be saved for use with GoTo commands and Paths. Trigger a save location b pressing the button on the homepage. PAL will take a GPS coordinate of the current location. Proceed by naming the location and saving to the locations library.

### Use GoTo Commands

The saved key locations can now be used in GoTo Commands. Trigger a Goto command by pressing the button on the homepage. Then select the desired key location.

<Note>PAL will navigate directly to the location from its current location</Note>
