## MavCommander Overview

MavCommander is the user interface for controlling and monitoring your PAL platforms.

| Number | Name            | Description                                  |
| ------ | --------------- | -------------------------------------------- |
| 1      | Vehicle Tab     | Switch between PALs connected to the network |
| 2      | Connect Button  | Connect to selected PAL                      |
| 3      | Homepage        | Navigate to homepage                         |
| 4      | Data Management | Navigate to Data Management page             |

## Homepage Control

<div style={{ display: "flex", justifyContent: "center" }}>

| Number | Name                  | Description                                  |
| ------ | --------------------- | -------------------------------------------- |
| 1      | Stop All              | Deactivate all PALs connected on the network |
| 2      | Stop                  | Deactivate PAL currently selected in tab     |
| 3      | Activate / Deactivate | Push button to activate and deactivate PAL   |
| 4      | Bin Control           | Slider (Up / Down) to control hydraulic bin  |
