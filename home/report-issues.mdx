## Report an Issue

If you encounter any problems with your PAL platform, accessories, or software, please report them using one of the following methods:

### GitHub Issues

For technical issues, bug reports, or feature requests, please create an issue on our GitHub repository.

<Card title="Create GitHub Issue" icon="github" href="https://github.com/your-repo/issues">
  Submit detailed bug reports and feature requests
</Card>

### Support Email

For urgent issues or questions that require immediate attention, please contact our support team directly.

<Card title="Contact Support" icon="envelope" href="mailto:support@cyphra.com">
  Get help from our support team
</Card>

## Before Reporting

To help us resolve your issue quickly, please include:

- **Platform Model**: PAL Dump or PAL Flatbed
- **Software Version**: Current version of the software/firmware
- **Steps to Reproduce**: Detailed steps that led to the issue
- **Expected Behavior**: What you expected to happen
- **Actual Behavior**: What actually happened
- **Error Messages**: Any error messages or logs
- **Screenshots/Videos**: Visual evidence of the issue (if applicable)
